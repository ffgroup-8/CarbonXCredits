# Carbon X Credits - Project Status Tracker

A visual project management tool for tracking website development progress, built with pure HTML/CSS/JS.

## Live Tracker
https://ffgroup-8.github.io/CarbonXCredits/

## Features
- **Sitemap Visualization** - Tree view of all website pages
- **Status Tracking** - Track progress across 6 categories (Design, Dev, SEO, Content, Sanity, Created)
- **Progress Rings** - Animated SVG ring charts showing overall completion
- **Dark Mode** - Toggle between light and dark themes
- **Content Strategy** - SEO/AEO planning and content creation tracking

## Status Categories
| Category | Description |
|----------|-------------|
| Design | Visual design and UI/UX |
| Dev | Frontend/backend development |
| SEO | Search engine optimization |
| Content | Content input and migration |
| Sanity | Sanity CMS setup and content modeling |
| Created | Final content approval |

## How to Update
1. Open `index.html` in your editor
2. Find the relevant page node
3. Change `not-started` to `completed` or `in-progress` on the appropriate status pill
4. Save and push to GitHub

## Development
- Pure HTML/CSS/JavaScript - no build tools required
- Deploy via GitHub Pages
- Single file deployment (`index.html`)
